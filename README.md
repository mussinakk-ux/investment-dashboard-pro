# 投資總控 Pro v4 Professional Edition（純投資版）

已移除：家庭資產、股息中心。

功能：
- 首頁 Dashboard
- 快速交易：買進/賣出自動更新庫存、平均成本、現金
- 手續費與證交稅自動計算
- 交易日期自動帶今天
- 已實現 / 未實現損益
- 00631L 操作中心
- 價差損益分析
- 退休中心
- 每日資產紀錄
- 備份 / 還原 JSON

部署：
1. 上傳 app.py、requirements.txt、.streamlit/config.toml 到 GitHub 覆蓋
2. Streamlit Cloud 重新部署
3. 預設密碼 1688，可到 Streamlit Secrets 修改
